# Подсказки по командной строке

**Команда смена директории**
```sh
cd  c:\<folder_name>
```

**Команда отображения текущей директории в MacOS**
```sh
pwd
```

**Листинг текущей директории**

для Windows
```sh
dir
```
для MacOS и Linux
```sh
ls
```

**Удаление файла**

в Windows
```sh
del <filename>
```
для MacOS Linux
```sh
rm <filename>
```
